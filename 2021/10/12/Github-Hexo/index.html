利用Github+Hexo搭建个人主页

发表于 2021-10-12
本文字数： 1.1k 阅读时长 ≈ 1 分钟

在学习中遇到问题便到互联网上搜索解决方案是我学习的常态，很长一段时间以来我都是游荡于各种大佬的社区主页和个人博客之中，于是也萌生了创建自己的个人主页的想法，便将它付诸实际。

互联网上已有成熟的搭建博客的流程，而我也借鉴了其中的一种方法，详情请点击这里，本文主要对该链接中的部分过时操作提出更新办法以及对我在此过程中遇到的部分问题进行阐述和解决。

1、注册Github账号

按照流程一步步走下来即可，注意可能会遇到部分邮箱无效的情况，建议使用Gmail账号进行注册。同时最近Github使用国内网络访问略有不稳定，有些浏览器也会受到插件、扩展的影响，如有条件，建议在科学上网的条件下使用Chrome浏览器进行注册。

2、安装Git

此步骤按照教程即可。

3、绑定Github并提交文件

绑定Github按照教程即可，提交文件时多数人都属于第一种情况，即本地没有git仓库。到输入
$ git push origin master后并不会出现如教程中所示的图片（即下图），而是弹出一个将Github账号与当前网页绑定的提示，同意即可。

6、解析域名

笔者是在万网上购买的域名，添加解析记录之后自动连接到了阿里云的dns服务器上，但后续出现有多台设备无法访问该域名的情况，在进行将该域名添加到科学上网中的直连状态的测试后，发现可以访问该域名，于是将域名的dns更改到cloudflare的服务器上，问题遂解决。
